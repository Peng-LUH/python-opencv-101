# python-opencv-101
Learning Image Processing with Python OpenCV
* [Quickstart](https://docs.opencv.org/4.x/db/dd1/tutorial_py_pip_install.html)
* [Examples](https://docs.opencv.org/4.x/examples.html)

## Initialization
```bash
uv init
uv run main.py
source .venv/bin/activate
```
## Install OpenCV
Install `opencv` using `uv add`.

```bash
uv add opencv-contrib-python
```
